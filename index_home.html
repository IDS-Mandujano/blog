AWS significa Amazon Web Services, y se refiere a una plataforma de servicios en la nube ofrecida por Amazon.com. AWS proporciona una amplia gama de servicios de computación en la nube, almacenamiento, bases de datos, aprendizaje automático, análisis, Internet de las cosas (IoT), seguridad, redes, desarrollo de aplicaciones y más. Estos servicios permiten a las empresas y a los individuos acceder a recursos informáticos de manera escalable y flexible sin la necesidad de invertir en infraestructura física.

  • Amazon EC2 (Elastic Compute Cloud): Ofrece capacidad de cómputo escalable en la nube.
  • Amazon S3 (Simple Storage Service): Proporciona almacenamiento en la nube escalable y duradero.
  • Amazon RDS (Relational Database Service): Ofrece servicios de bases de datos relacionales gestionadas.
  • AWS Lambda: Permite ejecutar código sin tener que provisionar o administrar servidores.
  • Amazon SNS (Simple Notification Service): Facilita la creación y envío de mensajes
  • Amazon DynamoDB: Un servicio de base de datos NoSQL completamente gestionado.
  • Amazon Redshift: Un servicio de almacenamiento de datos completamente gestionado y optimizado para realizar consultas SQL complejas.
  • Amazon VPC (Virtual Private Cloud): Proporciona una red privada virtual en la nube.
